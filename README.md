MongoMailStore
==============

Rails plugin for catch outcoming emails and store in MongoDB

Requirements
=======

  MongoMapper gem

Usage
=======

  make a proper connection to mongo_mapper

  install plugin

  u are done! :)
  
    the plugin copies a mongo_mail_store.yml in /config for set the store option for the environments
